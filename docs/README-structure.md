# E-CLI Documentation Structure

- `README.md` — User-facing quickstart, install, config, and CLI reference
- `docs/architecture.md` — Contributor architecture overview, component descriptions, and file layout
- `docs/safety-model.md` — Safety policy, classes, approval gate, audit log, and trusted commands
- `docs/skill-authoring.md` — Guide for third-party skill/plugin authors (manifest schema, entrypoint, OS variants, knowledge URLs)
- `docs/bundled-runtime.md` — Bundled helper model, asset download/verification, and runtime lifecycle
- `docs/offline-mode.md` — Air-gapped and fully offline deployment guide
- `docs/connected-mode.md` — Anthropic Claude setup, provider fallback chain, and MCP server configuration
- `docs/troubleshooting.md` — Common issues, diagnostic commands, and solutions
