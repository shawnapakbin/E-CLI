# E-CLI Documentation Structure

- `docs/architecture.md`: Contributor architecture overview
- `docs/safety-model.md`: Safety policy, classes, and enforcement
- `docs/skill-authoring.md`: Guide for third-party skill/plugin authors
- `docs/bundled-runtime.md`: Bundled helper model and runtime
- `docs/offline-mode.md`: Air-gapped and offline deployment
- `docs/connected-mode.md`: Cloud/remote provider configuration
- `docs/troubleshooting.md`: Common issues and solutions
